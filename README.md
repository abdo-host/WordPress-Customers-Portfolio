## WordPress Customers Portfolio
#### WordPress plugin for building your customer portfolio

Built on PHP composer and React.js plugin


#### Install Dependencies
```
npm install
```
#### Start Developing
```
npm start
```
#### Build and Minify
```
npm run build
```
